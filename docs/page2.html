    コンパイル

    Ctrl + Shift + @でターミナルを開き、次のコマンドを実行してみましょう。

    rustc src/main.rs

    実行すると、「main」という実行ファイルが生成されるはずです。
    この作業がコンパイルです。コンパイルとは、私たちが書いたソースコード(ここではmain.rs)を、 コンピュータが理解できるバイナリコードに変換する作業です。


    実行

    次に実行ファイルを実行してみましょう。
    ターミナルで次のコマンドを実行してみてください。

    ./main

    実行すると、ターミナルに「Hello, world!」と出力されるはずです。


    Cargoを使う

    ここまでは２つのコマンドを使い、ソースコードのコンパイルと実行を行いました。
    ここからは、もう１つの方法を紹介します。

    Cargoとは

    CargoとはRustのビルドシステム兼パッケージマネージャです。
    これはRustをインストールすると、付属してきます。
    これを使うことで、コードのビルドと実行、さらにはライブラリのインストールとビルドまでを
    簡単におこなうことができます。


    Cargoを使ってビルド

    実際にCargoを使ってコードのビルドをおこなってみましょう。
    ターミナルで次のコマンドを実行してみてください。

    cargo build

    実行すると、target/debug内にプロジェクトと同じ名前の実行ファイルが生成されるはずです。ちなみに

    cargo check

    というコマンドもあります。これは実行ファイルを生成せずにプロジェクトのコンパイルをおこなうため、 cargo buildよりも高速です。プログラムのエラーチェックなどに使えます。


    Cargoを使って実行

    ターミナルで次のコマンドを実行してみてください。

    cargo run

    このコマンドを使うことで、ソースコードのコンパイルから実行までをおこなうことができます。
