    • -環境構築-

    この章では、Rustの学習に必要な環境構築について紹介しています


    1.Ubuntuの取得

    マイクロソフトストアを起動し、「Ubuntu」と検索する

    Ubuntuをインストールする

    ※このサイトでは「Ubuntu 20.04.4 LTS」を使用しています。

    2.WSLのインストール

    1.コマンドプロンプトを管理者モードで起動し、次のコマンドを実行する

    wsl --install


    2.Windowsを再起動する


    3.再起動後、Ubuntuを起動しUNIXアカウントを作成する


    4.次のコマンドを実行する

    sudo apt update

    sudo apt dist-upgrade

    sudo apt autoremove


    5.次のコマンドを実行し、build-essentialをインストールする

    sudo apt-get update

    sudo apt-get install build-essential


    2.VScodeのインストール

    コードエディタとして、ここではVisual Studio Codeを使用します。

    1.次のサイトからVScodeのインストーラーを入手する

    インストーラーの入手

    2.インストーラーを起動し、VScodeをインストールする

    インストール中にでてくる「PATHへの追加」のチェックは必ずしてください。


    3.VScodeを起動せずに、インストーラーを終了する


    4.Windowsを再起動する


    3.Rustのインストール

    1.Ubuntu上で次のコマンドを実行する

    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

    2.インストール方法を聞かれるので、1を選択する


    3.次のコマンドを実行する(パスを通す)

    source $HOME/.cargo/env


    4.次のコマンドでRustのバージョンを確認する

    rustc --version

    cargo --version

    インストールが正しくおこなわれていれば、バージョンが表示されるはずです。


    4.作業用ディレクトリの作成

    1.次のコマンドを実行し、作業用ディレクトリを作成する

    mkdir $HOME/work


    2.次のコマンドを実行し、作成した作業用ディレクトリに移動する

    cd $HOME/work


    3.次のコマンドでRustのプロジェクトを作成する

    cargo new --bin rust_training

    プロジェクト名は適宜変更してください


    4.次のコマンドで、作成したプロジェクトディレクトリに移動する

    cd $HOME/work/rust_training


    5.VScodeの設定

    1.プロジェクトディレクトリ直下にいる状態で、次のコマンドを実行する(VScodeの起動)

    code .

    起動したとき、「Do you truet the...」という警告が出るかもしれませんが、「Yes」を選択してください。

    2.拡張機能「Remote - WSL」をインストールする


    3.左下の緑色の矢印をクリックする


    4.「Reopen Folder in WSL」を選択する

    5.Ctrl + @でターミナルを起動し、bashになっていることを確認する


    6.拡張機能「Rust」をインストールする

    インストールしmain.rsを開くと、右下に「Some Rust Components not installed. install?」と
    ポップアップが表示されるので、「Yes」を選択してください。
